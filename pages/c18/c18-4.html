18.4 其它伪类选择器
1.否定选择器

对选择器取反的选择器，语法:

:not(选择器)
2.:empty选择器

:empty选择器匹配没有定义任何子元素的元素。

<style> div:empty{ border: red solid thin; } </style> <div style="border: black solid thin">outer div<div></div></div>
outer div
3.:lang选择器
<style> :lang(en){ color:red; } </style> <a lang="en-us">en-us</a> <a lang="en-uk">en-uk</a> en-us en-uk

可以看到，它和[lang|="en"]选择器结果一样。

4.:target选择器

表示URL片段标识符指向的元素。一般URL指向一个页面，但带上#id可以指向某个元素，这个被指向的元素会被:target选择。但是不能跨页面选择哦。

<style> :target{ border: thin black solid; padding:4px; color:red; } </style> <a href="#a">visit a</a> <a href="#b">visit b</a> <a href="../c16/c16-3.html#target">跨页面</a> <a id="a">本页a</a> | <a id="b">本页b</a> visit a visit b 跨页面 本页a | 本页b
