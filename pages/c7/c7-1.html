7.1 构建基本的文档结构
前言

最基本的HTML5元素:文档元素和元数据元素。

1.DOCTYPE元素

DOCTYPE元素独一无二自成一类，每个HTML文档都必须以DOCTYPE元素开头(虽然不以它开头也可能不会出错)。

表1 DOCTYPE元素
元素 DOCTYPE
元素类型 无
允许具有的父元素 无
局部属性 无
内容 无
标签用法 单个开始标签
是否为HTML5新增 否
在HTML5中的变化 HTML4中要求要有的DTD已不再使用
习惯样式 无

使用：

<!DOCTYPE html>

我的疑惑：局部属性：无，这里的html不是一个属性吗？并且前面的这个叹号又表示什么？在w3School看了一下，那里他们认为!是标签的一部分。

2.html元素
表2 html元素
元素 html
元素类型 无
允许具有的父元素 无
局部属性 mainfest
内容 head元素和body元素各一
标签用法 开始标签和结束标签，内含其它元素
是否为HTML5新增 否
在HTML5中的变化 新增mainfest属性，html4版本中的属性不再使用
习惯样式 html{display:block;} html:focus{outline:none;}

使用：

<!DOCTYPE html> <html> ...... </html>
3.head元素
表3 head元素
元素 head
元素类型 无(为什么不是流元素?)
允许具有的父元素 html
局部属性 无
内容 必须有一个title元素，其它元数据元素可有可无
标签用法 开始标签和结束标签，内含其它元素
是否为HTML5新增 否
在HTML5中的变化 无
习惯样式 无

使用：

<!DOCTYPE html> <html> <head> <title>......</title> </head> </html>
4.body元素
表4 body元素
元素 body
元素类型 无(为什么不是流元素?)
允许具有的父元素 html
局部属性 无
内容 所有短语元素和流元素
标签用法 开始标签和结束标签
是否为HTML5新增 否
在HTML5中的变化 alink,background,bgcolor,link,margintop,marginbottom,marginleft,marginright,marginwidth,text和vlink属性已不再使用。这些属性的效果可用CSS实现
习惯样式 body{display:block;margin:8px;} body:focus{outline:none;}

使用：

<!DOCTYPE html> <html> <head> <title>Example</title> </head> <body> <p>abc</p> <a href="http://xxx.ccc">visit xxx</a> </body> </html>
