7.2 用元数据元素说明文档

元数据元素可以用来提供关于HTML文档的信息。它们本身不是文档的内容，但提供了关于后面的文档内容的信息。元数据元素应该放在head元素中。

1.设置文档标题

title元素

表1 title元素
元素 title
元素类型 元数据
允许具有的父元素 head
局部属性 无
内容 文档标题或对文档内容言简意赅的说明
标签用法 开始标签和结束标签，内含文字
是否为HTML5新增 否
在HTML5中的变化 无
习惯样式 title{display:none;}
2.设置相对URL的解析基准

base元素可以用来设置一个基准URL，让HTML文档中的相对链接在此基础上进行解析。

表2 base元素
元素 base
元素类型 元数据
允许具有的父元素 head
局部属性 href、target
内容 无
标签用法 虚元素形式
是否为HTML5新增 否
在HTML5中的变化 无
习惯样式 无
2.1 使用href属性
<!DOCTYPE html> <html> <head> <base href="http://abc.com/asset"> </head> <a href="http://abc.com/asset/img/1.png">图片1</a> <a href="img/1.png">图片1</a> </html>
2.2 使用target属性

target属性的作用是告诉浏览器如何打开URL。后面章节会介绍这个属性。

在HTML5中的变化 charset是html5中新增的。在HTML4中，http-equiv属性可以有任意多个不同的值。而在HTML5中情况有所不同，只有本小节所说的值才能用。在HTML4中的scheme属性在HTML5中已不再使用。 此外，现在已不再使用meta元素来指定网页所使用的语言。
